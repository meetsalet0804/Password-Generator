# 🔐 Password Generator

A simple and responsive Password Generator web application built using **HTML, CSS, and JavaScript**. This project helps users generate strong and secure passwords with customizable options.

---

## 📖 Project Overview

The Password Generator is designed to improve online security by creating random and strong passwords. Users can choose the password length and include uppercase letters, lowercase letters, numbers, and special symbols to generate a secure password.

---

## ✨ Features

* 🔐 Generate strong random passwords
* 📏 Select custom password length
* 🔠 Include uppercase letters
* 🔡 Include lowercase letters
* 🔢 Include numbers
* 🔣 Include special symbols
* 📋 One-click password copy
* 💻 Simple and user-friendly interface
* 📱 Responsive design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* GitHub
* GitHub Pages

---

## 🚀 How to Use

1. Open the Password Generator.
2. Enter the desired password length.
3. Select the required password options.
4. Click the **Generate Password** button.
5. Copy the generated password using the **Copy Password** button.
